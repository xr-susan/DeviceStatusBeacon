# 返回代码

## 命令行模式 exitCode
| exitCode | 说明                      |
| :------- | :----------------------- |
| -1       | 发生异常                  |
| 0        | 成功                      |
| 1        | 相应对象已存在，无法创建    |
| 2        | 相应对象不存在，无法执行    |
| 3        | 参数错误                  |
| 4        | 数据超限                  |
| 5        | 暂不支持/暂未实现          |
